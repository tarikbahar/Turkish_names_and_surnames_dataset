# Türkçe isim ve soyisimler listesi
Bu repo Türkçe erkek, kadın ve unisex isimleri ile birlikte Türkçe soyisimleri içermektedir.

Turkce_isimler_listesi.txt dosyasındaki;
k = kadın ismi olduğu,
e = erkek ismi olduğu,
u = unisex isim olduğu anlamına gelmektedir.



# Turkish names and surnames lists
This repository contains Turkish female, male, unisex names and surnames.
Turkce_isimler_listesi.txt file contains Turkish names.
k means = female,
e means = male,
u means = unisex names.

Turkce_soyisimler_listesi.txt file contians Turkish surnames.
